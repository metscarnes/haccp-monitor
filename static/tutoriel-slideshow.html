Pourquoi étalonner ?

Pour s'assurer que le thermomètre donne la bonne température. C'est obligatoire (HACCP) et c'est une question de sécurité.

  • Préparez un récipient avec eau + glace. Attendez 2 minutes : c'est 0°C.
  • Plongez le thermomètre, attendez qu'il se stabilise, tapez la valeur lue.
  • Si l'écart est ≤ ±1°C → conforme ✅. Tapotez « Valider ».
⚠️ Si l'écart est > 1°C : recommencez (mauvais mélange ?). Si l'écart persiste, prévenez le responsable — le thermomètre doit être remplacé.

🖨️ IMPRIMANTE — Brother QL-820NWB SLIDE 23

Au quotidien, vérifier le voyant

🟢
Vert
Tout va bien, prête à imprimer.
🟠
Orange clignotant
Plus de rouleau, ou bourrage. Ouvrez le capot, retirez l'étiquette coincée.
🔴
Rouge fixe
Erreur. Éteignez-la, attendez 10 sec, rallumez. Sinon, appelez le responsable.

Changer le rouleau d'étiquettes

  1. Ouvrez le capot (bouton sur le côté).
  2. Retirez l'ancien support en tirant doucement vers le haut.
  3. Placez le nouveau rouleau, la sortie de l'étiquette vers l'avant.
  4. Fermez le capot jusqu'à entendre un clic.
  5. Refaites un essai d'impression depuis le Stock ou le Calendrier DLC.

« Température hors plage »
L'app vous protège. Reprenez la mesure, vérifiez le thermomètre. Si l'écart est réel, refusez le produit.
🖨️ L'imprimante n'imprime pas — Procédure d'extinction/rallumage
  1. Appui long sur le bouton d'allumage pendant 2 secondes.
  2. La LED passe à l'🟠 orange, puis l'imprimante s'éteint complètement.
  3. Ré-appuyez sur le bouton pendant 2 secondes pour la rallumer.
  4. Attendez 45 secondes que l'icône 📶 Wi-Fi arrête de clignoter (lumière fixe = prête).
  5. Relancez une réimpression depuis le Stock ou le Calendrier DLC.
Si toujours rien après cette procédure → voyant rouge/orange persistant ou pas de Wi-Fi → prévenez le responsable.
